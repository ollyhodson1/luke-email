# GitHub Pages deployment folder

This folder is ready for GitHub Pages.

In your GitHub repository, go to:

Settings → Pages → Build and deployment → Deploy from a branch

Then choose:

- Branch: `main`
- Folder: `/docs`

The app will then run directly from `docs/index.html`.

## Unread reply behaviour

After students send the email, the app stays on the same screen. Becca Richardson’s reply arrives after 20 seconds as one unread Inbox message.
